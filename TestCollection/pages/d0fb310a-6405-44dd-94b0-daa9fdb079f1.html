Research interests

I have broad interests in computer and robotic vision, machine learning, probabilistic graphical models, and optimization. My main research is in the application of machine learning techniques (specifically, conditional Markov random fields) to geometric and semantic scene understanding.

Available student projects

Image-completion with user-assisted sketching

A number of algorithms have been proposed for the task of completing missing portions of an image. In this work we examine the problem of guiding the completion of an image by providing a user sketch. The project is suitable for an undergraduate or masters thesis project. 

 Interactive 3D reconstruction from monocular images and video

This project aims to allow users to reconstruct 3D scenes from images and videos by providing minimal cues to the algorithm, for example, marking a few pixels as foreground or background.

Learning with structured representations in a dataflow framework

Implement software for optimizing learning objectives over structured datatypes. The project builds on infrastructure developed in the DARWIN software library (http://drwn.anu.edu.au). Before taking the project students should download and install this library. A strong background in C/C++ is required. 

  Large-Scale Object Detection and Scene Understanding

The availability of large volumes of data is changing the way many machine learning and computer vision problems are tackled. Data-driven approaches are achieving state-of-the-art performance in the new world of Big Data. In this project students will invesitgate problems in large-scale object detection and scene understanding with the aim of submitting algorithms to one or more of the various computer vision challenges (e.g., PASCAL VOC, ImageNet, etc). Students should have a background in machine learning and/or computer vision and strong programming skills.
